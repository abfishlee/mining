🔥 작업형 제1유형: 전처리 (각 10점, 총 30점)

"제시된 데이터의 특성을 파악하고, 조건에 맞게 필터링 및 변환하여 단일 숫자(예: 12.34)를 print() 함수로 출력해야 합니다."

[문제 1-1] `titanic.csv` 데이터에서 요금('Fare')의 상위 10%에 해당하는 값을 구하고, 요금이 그 값 이상인 데이터들의 나이('Age') 평균을 소수점 둘째 자리까지 출력하시오. (결측치는 전체 나이 평균으로 대체할 것)

import pandas as pd

df = pd.read_csv('../data/titanic.csv')

# 1. 결측치 처리
df['Age'] = df['Age'].fillna(df['Age'].mean())

# 2. 상위 10% 요금 값 구하기 (quantile 0.9)
top_10_fare = df['Fare'].quantile(0.90)

# 3. 요금이 상위 10% 이상인 데이터 필터링
filtered_df = df[df['Fare'] >= top_10_fare]

# 4. 나이 평균 구하기 및 출력 (소수점 둘째자리 반올림)
result = round(filtered_df['Age'].mean(), 2)
print(result)

[문제 1-2] 'SibSp'와 'Parch' 값을 더하여 'Family' 변수를 만들고, 전체 데이터의 80%만 추출( 앞에서부터 자르기 )한 후, 'Family' 변수의 분산(Variance)을 구하시오. (소수점 버림 처리)

import pandas as pd
import math

df = pd.read_csv('../data/titanic.csv')

df['Family'] = df['SibSp'] + df['Parch']

# 앞에서부터 80% 데이터 추출
cut_idx = int(len(df) * 0.8)
df_80 = df.iloc[:cut_idx]

# 분산 구하기 및 버림(math.trunc)
result = math.trunc(df_80['Family'].var())
print(result)

🔥 작업형 제2유형: 머신러닝 모델링 (총 40점)

"제공된 훈련용(train) 데이터로 모델을 만들고, 평가용(test) 데이터의 '생존할 확률(probability)'을 예측하여 CSV 파일로 제출해야 합니다. 평가지표는 ROC-AUC 입니다."

[문제 2] `titanic.csv`를 불러와서 앞쪽 700개를 train 데이터로, 나머지 191개를 test 데이터로 임의로 나누어 사용하세요. 결측치를 처리하고, 범주형 변수를 인코딩한 후, 랜덤포레스트 모델을 활용해 test 데이터의 탑승객별 생존 '확률'을 구하여 result.csv 파일로 저장하시오.

import pandas as pd
from sklearn.ensemble import RandomForestClassifier

df = pd.read_csv('../data/titanic.csv')

# 시험용 데이터 분할 가상 환경 세팅
train = df.iloc[:700].copy()
test = df.iloc[700:].copy()

# 1. 결측치 처리 (Test에도 Train의 통계값을 사용해야 Data Leakage가 아님!)
age_mean = train['Age'].mean()
train['Age'] = train['Age'].fillna(age_mean)
test['Age'] = test['Age'].fillna(age_mean)

# 2. 범주형 변수 처리 (원핫 인코딩) - 겹치는 열만 사용
features = ['Pclass', 'Sex', 'Age', 'Fare']
X_train = pd.get_dummies(train[features], drop_first=True)
y_train = train['Survived']
X_test = pd.get_dummies(test[features], drop_first=True)

# 3. 모델 학습 및 예측 (생존 '확률' 이므로 predict_proba 사용)
model = RandomForestClassifier(random_state=42)
model.fit(X_train, y_train)

# 생존(1)할 확률은 두 번째 열([:, 1])에 있습니다.
pred_proba = model.predict_proba(X_test)[:, 1]

# 4. 제출용 DataFrame 생성 및 저장
# 시험에서는 주로 PassengerId 같은 고유 식별자와 확률을 저장하도록 요구합니다.
submission = pd.DataFrame({
    'PassengerId': test['PassengerId'],
    'Survived_Probability': pred_proba
})
submission.to_csv('result.csv', index=False)
print("result.csv 저장 완료!")

🔥 작업형 제3유형: 가설 검정 (각 15점, 총 30점)

"새롭게 추가된 유형입니다. 통계적 유의성을 검정하고 검정통계량 및 p-value를 도출해야 합니다."

[문제 3] `titanic.csv`에서 'Pclass'가 1등급인 사람들의 평균 'Fare(요금)'가 80달러라고 할 수 있는지 단일표본 T-검정(One-sample T-test)을 수행하고, p-value를 반올림하여 소수점 4자리까지 출력하시오.

import pandas as pd
from scipy import stats

df = pd.read_csv('../data/titanic.csv')

# 1등급(Pclass==1) 승객의 요금만 추출
pclass1_fare = df[df['Pclass'] == 1]['Fare'].dropna()

# 단일표본 T-검정 수행 (모집단 평균이 80인지 검정)
t_stat, p_val = stats.ttest_1samp(pclass1_fare, 80)

print(round(p_val, 4))
# 결과가 0.05보다 작다면 평균 요금은 80달러가 아니라고 결론 내릴 수 있습니다.
