Pandas 전처리/EDA 실전 패턴

기초 문법을 알았다면 이제 시험 지문을 Pandas 코드로 바꾸는 훈련이 필요합니다. 이 페이지는 조건, 집계, 정렬, 교차표, 파생변수처럼 작업형 1과 ADP EDA에서 자주 반복되는 패턴을 정리합니다.

1. 조건 필터링은 질문을 행으로 번역하는 일

"여성이고 1등급이고 생존한 승객"처럼 지문에 조건이 여러 개 나오면 각 조건을 괄호로 감싸고 &로 연결합니다. 파이썬의 and가 아니라 Pandas의 벡터 조건 연산자를 써야 합니다.
female_first_survived = df[
    (df['Sex'] == 'female') &
    (df['Pclass'] == 1) &
    (df['Survived'] == 1)
]
print(len(female_first_survived))

2. 집계는 "무엇별로, 무엇을, 어떻게"로 읽기

등급별 평균
df.groupby('Pclass')['Fare'].mean()
무엇별로: Pclass, 무엇을: Fare, 어떻게: mean
요일별 최대 집단 찾기
tips.groupby('day')['total_bill'].mean().idxmax()
평균이 가장 큰 그룹 이름을 찾습니다.
빈도 세기
df['Embarked'].value_counts()
범주형 변수의 분포와 최빈값을 확인합니다.
교차표
pd.crosstab(df['Sex'], df['Pclass'])
두 범주형 변수가 어떻게 만나는지 표로 봅니다.

3. 정렬과 상위값은 기준 열을 먼저 확인

"가장 큰", "상위", "하위"가 나오면 정렬 기준과 오름차순/내림차순을 먼저 정합니다. ascending=False는 큰 값부터 보는 뜻입니다.
top_row = df.sort_values('Fare', ascending=False).iloc[0]
print(top_row['PassengerId'])
  • sort_values() 뒤에 iloc[0]을 쓰면 정렬 후 첫 행을 가져옵니다.
  • 동점 처리 지시가 있으면 sort_values(['Fare', 'Age'], ascending=[False, True])처럼 기준을 추가합니다.

4. 파생변수는 문제 풀이를 쉽게 만드는 중간 계산

실전에서는 원래 컬럼 그대로 답이 나오지 않는 경우가 많습니다. 가족 수, 비율, 로그 변환, 결제 단가처럼 새 컬럼을 만들어야 지문을 코드로 옮기기 쉬워집니다.
tips['tip_rate'] = tips['tip'] / tips['total_bill'] * 100
print(round(tips['tip_rate'].mean(), 2))

5. 실전 풀이 체크리스트

  • 지문에서 데이터셋, 조건, 대상 컬럼, 집계 함수, 출력 형식을 표시합니다.
  • 조건 필터링 후에는 len()이나 head()로 중간 결과가 맞는지 확인합니다.
  • 정답 출력 전에는 round(), int(), floor() 같은 출력 형식을 다시 봅니다.
  • 문제은행에서는 먼저 시간 제한 없이 풀고, 두 번째부터 타이머를 켭니다.
