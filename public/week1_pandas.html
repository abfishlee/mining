🛒 1. 데이터 불러오기 (장보기)

"판다스의 read_csv()는 마트에서 장을 봐서 도마(DataFrame) 위로 재료를 올려놓는 과정입니다."

파이썬을 열고 제일 먼저 해야 할 일은 판다스를 불러오고 데이터를 가져오는 것입니다.

import pandas as pd

# 타이타닉 데이터를 df라는 이름의 도마(DataFrame)에 올립니다.
df = pd.read_csv('../data/titanic.csv')

# 도마 위의 재료 중 위쪽 5개만 슬쩍 맛봅니다.
print(df.head())

🔍 2. 데이터 상태 확인 (재료 신선도 체크)

"요리를 시작하기 전, 재료가 몇 개나 있는지, 썩은 부분(결측치)은 없는지 매의 눈으로 확인합니다." 
# 전체 재료의 행 개수와 빈칸 유무, 데이터 타입을 요약해서 보여줍니다.
df.info()

# 열(Column)별로 빈칸이 몇 개씩 있는지 정확히 세어봅니다.
print(df.isnull().sum())

🔪 3. 결측치 처리 (썩은 채소 잘라내기/새살 채우기)

"나이(Age) 정보가 없는 빈칸이 많네요. 이 빈칸들을 사람들의 '평균 나이'로 꾹꾹 채워 넣어 봅시다." 
# 나이의 평균을 구합니다.
mean_age = df['Age'].mean()

# fillna() 함수를 이용해 빈칸을 평균값으로 덮어씁니다.
df['Age'] = df['Age'].fillna(mean_age)

[문제 1] df에서 아래쪽 끝 3개의 데이터를 슬쩍 맛보세요. (힌트: tail)

[문제 2] df의 'Fare'(요금) 열만 따로 떼어서 요금의 "평균"을 구해보세요.

print(df['Fare'].mean())

[문제 3] df의 'Cabin'(객실 번호) 열은 결측치(빈칸)가 너무 많습니다. 이 열을 도마에서 완전히 버리세요. (힌트: drop)

# axis=1은 '열(Column)'을 버리겠다는 의미입니다.
df = df.drop('Cabin', axis=1)

[문제 4] 'Survived'(생존 여부) 열을 기준으로, 생존자(1)와 사망자(0)가 각각 몇 명인지 세어보세요.

print(df['Survived'].value_counts())
