데이터 양념하기: 전처리 심화

기계학습 모델이 데이터를 더 잘 소화할 수 있도록 형태를 바꿔봅시다! 🧂

📏 1. 데이터 크기 맞추기 (스케일링, Scaling)

"몸무게는 킬로그램(kg)이고 키는 센티미터(cm)라면 숫자의 크기 차이가 너무 큽니다. 기계는 숫자가 크면 더 중요하다고 착각합니다. 모든 숫자를 0에서 1 사이로 통일해주는 것이 MinMaxScaler입니다." 
import pandas as pd
from sklearn.preprocessing import MinMaxScaler

# 1주차에 쓰던 타이타닉 데이터를 다시 부릅니다.
df = pd.read_csv('../data/titanic.csv')

# 나이와 요금의 척도를 0~1 사이로 줄여봅시다!
scaler = MinMaxScaler()
df[['Age', 'Fare']] = scaler.fit_transform(df[['Age', 'Fare']])

# 이제 요금이 50달러이던 것이 0.23 처럼 작은 비율로 바뀌었습니다.
print(df[['Age', 'Fare']].head())

💼 현업 선배의 실무 노하우 (실수 방지!)

🚨 Data Leakage (데이터 누수)의 함정을 피하세요!

초보자, 심지어 실무자들도 가장 많이 하는 치명적인 실수가 바로 "전체 데이터를 한 번에 스케일링하는 것"입니다. 시험 문제(Test Data)는 모델이 미리 알면 안 되는 미지의 영역입니다.

  • 나쁜 예: 전체 데이터를 fit_transform() 해버림. (미래의 정보를 끌어다 쓰는 컨닝!)
  • 실무 정석: 학습용 데이터(Train)로만 fit()을 해서 기준(최소/최대값 등)을 잡은 뒤, 테스트용 데이터(Test)는 기준만 적용하는 transform()을 사용해야 합니다.

🔤 2. 문자를 숫자로 번역하기 (인코딩, Encoding)

"컴퓨터는 'male', 'female' 같은 영어를 모릅니다. 무조건 0과 1로 번역해줘야 합니다. 외국어 라벨을 숫자로 바꿔주는 이 과정을 Label Encoding이라고 합니다." 
from sklearn.preprocessing import LabelEncoder

encoder = LabelEncoder()
# 성별(Sex) 열의 'male', 'female'을 0과 1로 변환합니다.
df['Sex'] = encoder.fit_transform(df['Sex'])

print(df[['Name', 'Sex']].head())

✨ 3. 새로운 요리법 창조 (파생 변수 생성)

"설탕과 간장을 섞어 '데리야키 소스'를 만들 듯, 기존 데이터들을 조합해 더 강력한 의미를 가진 새로운 기둥(Column)을 만들어내는 기술입니다." 
# 부모자식(Parch) 수와 형제자매(SibSp) 수를 더해서 '가족의 총 크기'를 만듭니다.
df['FamilySize'] = df['Parch'] + df['SibSp'] + 1 # 1은 본인

# 가족이 한 명도 없으면 1(True), 아니면 0(False)인 파생변수를 만듭니다.
df['IsAlone'] = (df['FamilySize'] == 1).astype(int)

print(df[['FamilySize', 'IsAlone']].head())

🚀 드릴 연습문제 (EDA 집중 훈련)

직접 주피터 노트북에 타이핑하며 손에 익혀보세요!

[문제 1] df의 'Pclass'(객실 등급: 1, 2, 3)는 이미 숫자지만, 사실은 순위를 나타내는 '범주형' 데이터입니다. 이 값을 원핫 인코딩(One-Hot Encoding)으로 바꾸기 위해 판다스의 pd.get_dummies()를 사용해 보세요.

# Pclass 1, 2, 3이 각각 독립된 열로 쪼개집니다.
df = pd.get_dummies(df, columns=['Pclass'])
print(df.head())

[문제 2] 'Embarked'(탑승 항구) 열의 빈칸(결측치)을 가장 많이 나오는 'S'로 채운 다음, 이 열을 LabelEncoder를 써서 0, 1, 2 숫자로 번역해 보세요.

df['Embarked'] = df['Embarked'].fillna('S')
encoder = LabelEncoder()
df['Embarked'] = encoder.fit_transform(df['Embarked'])

[문제 3] 'Fare'(요금) 열을 MinMaxScaler 대신 평균이 0, 표준편차가 1이 되는 StandardScaler를 사용해 정규화(Scaling) 해보세요.

from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
# 주의: scaler는 2차원 배열을 원하므로 df[['Fare']] 처럼 대괄호를 두 번 씁니다.
df[['Fare']] = scaler.fit_transform(df[['Fare']])

[문제 4: 새로운 데이터 등장!] 새롭게 추가된 tips.csv (레스토랑 팁 데이터)를 불러오세요. 'total_bill'(총 결제 금액) 대비 'tip'(팁)이 몇 퍼센트인지 나타내는 'tip_rate'라는 파생 변수를 만들어 보세요. (계산식: tip / total_bill * 100)

import pandas as pd

tips['tip_rate'] = (tips['tip'] / tips['total_bill']) * 100

[문제 5] tips.csv에서 'day'(요일: Thur, Fri, Sat, Sun) 열을 원핫 인코딩(get_dummies) 하세요. 이때 drop_first=True 옵션을 주어 다중공선성을 방지해 보세요! (ADP 실기 단골 팁)

# drop_first=True 를 하면 첫 번째 카테고리 열을 삭제합니다.
# n-1 개의 열만으로도 모든 카테고리를 구분할 수 있기 때문입니다!
tips_encoded = pd.get_dummies(tips, columns=['day'], drop_first=True)
print(tips_encoded.head())
