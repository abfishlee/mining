통계 기초 보강

통계 문제는 계산보다 선택과 해석에서 틀리는 경우가 많습니다. 이 페이지는 "어떤 검정을 왜 쓰는지", "p-value를 어떻게 말해야 하는지", "실무에서 어떤 한계를 써야 하는지"를 반복 가능한 형태로 정리합니다.

1. 검정 선택은 변수 타입과 집단 수로 결정

한 집단 평균 vs 기준값
예: 평균 tip이 3달러인가?
stats.ttest_1samp()
두 독립 집단 평균
예: 남녀 total_bill 평균 차이
stats.ttest_ind(..., equal_var=False)
세 집단 이상 평균
예: 요일별 tip 평균 차이
stats.f_oneway()
범주형 변수 관계
예: 요일과 식사 시간대의 관련성
pd.crosstab() + stats.chi2_contingency()
수치형 두 변수 관계
예: total_bill과 tip 관계
stats.pearsonr()
가정 점검
정규성: Shapiro, 등분산성: Levene
가정이 약하면 해석에 한계를 씁니다.

2. p-value 해석 문장 템플릿

p-value는 "귀무가설이 맞다고 가정했을 때, 지금 같은 데이터가 관측될 가능성"입니다. 작으면 귀무가설과 데이터가 잘 맞지 않는다는 뜻이지, 대립가설이 100% 증명됐다는 뜻은 아닙니다.
  • p-value < 0.05: 유의수준 0.05에서 귀무가설을 기각한다. 통계적으로 유의한 차이/관계가 있다고 볼 수 있다.
  • p-value >= 0.05: 귀무가설을 기각하지 못한다. 차이/관계가 있다고 보기 어렵다.
  • 주의: "귀무가설을 채택한다", "차이가 전혀 없다", "인과관계가 증명됐다"라고 쓰면 위험합니다.

3. 코드 패턴

import pandas as pd
from scipy import stats

tips = pd.read_csv('data/tips.csv')

male = tips[tips['sex'] == 'Male']['total_bill']
female = tips[tips['sex'] == 'Female']['total_bill']

t_stat, p_value = stats.ttest_ind(male, female, equal_var=False)
print(round(p_value, 4))
  • 두 집단이 서로 다른 사람/행이면 독립표본입니다.
  • 등분산을 확신하기 어렵다면 Welch t-test인 equal_var=False가 안전합니다.
  • ADP 답안에는 검정 선택 이유, 가설, 판단, 한계를 함께 씁니다.

4. 실무 해석에서 꼭 덧붙일 말

  • 통계적 유의성이 있어도 효과 크기가 실무적으로 충분한지는 별도 검토가 필요합니다.
  • 표본이 크면 작은 차이도 유의해질 수 있고, 표본이 작으면 큰 차이도 유의하지 않을 수 있습니다.
  • 상관관계는 인과관계가 아닙니다. 교란 변수와 실험 설계 여부를 확인해야 합니다.
  • 정규성/등분산성/독립성 가정이 약하면 비모수 검정 또는 강건한 검정을 고려합니다.
