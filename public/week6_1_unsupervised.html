Week 6-1: 비지도학습 & 연관 분석

정답 없는 데이터에서 숨겨진 보물(패턴) 찾기

🧭 0. 비지도학습은 언제 쓰는가?

비지도학습은 정답 라벨이 없을 때 데이터 내부의 구조를 찾는 방법입니다. 예측 정확도를 바로 계산하기 어렵기 때문에, 분석 목적과 해석 가능성이 특히 중요합니다.

목표 대표 기법 활용 예시 주의점
비슷한 대상을 묶고 싶다 K-Means, 계층적 군집화 고객 세분화, 제품 유형 분류, 지역 패턴 구분 정답이 없으므로 군집 해석과 검증이 중요
군집 수를 탐색하고 싶다 엘보우, 실루엣 점수, 덴드로그램 몇 개의 고객군으로 나눌지 판단 점수만 보지 말고 업무적으로 설명 가능한 K인지 확인
함께 발생하는 규칙을 찾고 싶다 연관규칙 분석 장바구니 분석, 교차판매, 추천 규칙 지지도만 높으면 흔한 상품 조합일 수 있음
비지도학습은 “정답률”이 없기 때문에 결과를 과신하기 쉽습니다. ADP에서는 군집별 특징, 군집 수 선택 근거, 실무 활용 가능성, 한계를 함께 써야 합니다.

🧩 1. K-Means와 실루엣 스코어 (Silhouette Score)

"K-Means로 나눈 조별 과제 그룹이 얼마나 똘똘 뭉쳐있는지(응집도), 다른 조원들과는 얼마나 멀리 떨어져 있는지(분리도)를 -1에서 1 사이의 점수로 평가합니다. 1에 가까울수록 완벽한 군집!"

거리 기반스케일링 필수K 사전 지정군집 해석 필요

K-Means는 각 데이터를 가장 가까운 중심점에 배정하고, 다시 중심점을 갱신하는 과정을 반복합니다. 거리 기반 알고리즘이므로 변수 단위가 다르면 큰 단위의 변수가 군집을 지배합니다.

K 선택법: 엘보우는 군집 내 오차가 급격히 줄어드는 지점을 보고, 실루엣 점수는 군집 내부 응집도와 군집 간 분리도를 함께 봅니다. 단, 점수가 가장 높은 K가 항상 실무적으로 좋은 K는 아닙니다. 
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
cluster = kmeans.fit_predict(X_scaled)

score = silhouette_score(X_scaled, cluster)
print("실루엣 점수:", score)

🌳 2. 계층적 군집화 (Hierarchical Clustering)

"가장 가까운 두 명을 먼저 짝지어주고, 그 다음 가까운 사람을 붙여주면서 거대한 토너먼트 대진표(Dendrogram)를 그리는 방식입니다. K(군집 개수)를 미리 정하지 않아도 되는 장점이 있습니다."

덴드로그램군집 수 탐색작은 데이터 해석 좋음

계층적 군집화는 가까운 데이터끼리 단계적으로 묶어 나갑니다. 덴드로그램을 보면 어떤 지점에서 군집을 자르는 것이 자연스러운지 시각적으로 판단할 수 있습니다.

연결 기준 의미 특징
single 가장 가까운 점끼리 거리 길게 이어지는 체인 형태가 생길 수 있음
complete 가장 먼 점끼리 거리 군집을 더 조밀하게 만드는 경향
ward 군집 내 분산 증가를 최소화 연속형 수치 데이터 군집화에서 자주 사용
계층적 군집화는 데이터가 커지면 계산량과 시각화 해석이 어려워집니다. 대규모 데이터에서는 K-Means로 빠르게 군집화하고, 샘플 데이터로 계층 구조를 보조 확인하는 식이 현실적입니다.

🛒 3. 연관 분석 (Association Rule Mining, Apriori)

"기저귀를 산 아빠들이 맥주도 같이 산다는 전설의 마케팅 분석법입니다! 'A를 사면 B도 산다'는 규칙을 찾아냅니다."

지지도(Support), 신뢰도(Confidence), 향상도(Lift)라는 3가지 지표를 이용해 규칙의 강도를 판단합니다. 특히 향상도(Lift)가 1보다 커야 진짜 의미 있는 연관성입니다.

지표 의미 주의사항
Support A와 B가 함께 등장하는 전체 비율 너무 낮으면 규칙이 희귀해서 활용하기 어려움
Confidence A가 있을 때 B도 있을 조건부 확률 B 자체가 흔하면 과대평가될 수 있음
Lift A가 B 발생을 얼마나 끌어올리는지 1보다 크면 양의 연관, 1이면 독립에 가까움
실무 활용: 교차판매, 패키지 구성, 추천 규칙, 진열 최적화에 사용합니다. 단, 연관규칙은 인과관계가 아니라 동시 발생 패턴입니다.

🧾 4. ADP 비지도학습 답안 작성 공식

본 문제는 정답 라벨 없이 데이터 내부의 유사한 구조를 파악하는 것이 목적이므로 비지도학습을 사용하였다.
거리 기반 군집화의 특성상 변수 단위 차이를 제거하기 위해 표준화를 수행하였다.
군집 수는 [엘보우/실루엣/덴드로그램] 결과와 해석 가능성을 함께 고려하여 선택하였다.
군집별 주요 변수 평균을 비교한 결과 [군집 특성]으로 해석할 수 있다.
다만 군집 결과는 정답이 아니며, 초기값과 변수 선택에 따라 달라질 수 있으므로 업무적 검증이 필요하다.

🚀 드릴 연습문제 (비지도학습 실전)

[문제 1: K-Means 기본] iris.csv에서 species를 제외하고 표준화 후 K-Means로 3개 군집을 만드세요. 실루엣 점수를 출력하고 군집별 평균을 비교하세요.

import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

iris = pd.read_csv('data/iris.csv')
X = iris.drop('species', axis=1)

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
iris['cluster'] = kmeans.fit_predict(X_scaled)

score = silhouette_score(X_scaled, iris['cluster'])
print('silhouette:', round(score, 4))
print(iris.groupby('cluster')[X.columns].mean())

[문제 2: K 선택] K를 2부터 6까지 바꾸며 실루엣 점수를 비교하세요. 가장 높은 점수의 K와, 해석 가능한 K가 반드시 같은지 생각해보세요.

for k in range(2, 7):
    model = KMeans(n_clusters=k, random_state=42, n_init=10)
    labels = model.fit_predict(X_scaled)
    score = silhouette_score(X_scaled, labels)
    print('k:', k, 'silhouette:', round(score, 4))

print('해석: 점수와 업무적 설명 가능성을 함께 고려해 K를 선택합니다.')

[문제 3: 연관규칙 수작업] 아래 거래 데이터에서 A를 샀을 때 B도 사는 규칙의 support, confidence, lift를 직접 계산하세요.

transactions = [
    {'A', 'B', 'C'},
    {'A', 'B'},
    {'A', 'C'},
    {'B', 'C'},
    {'A', 'B', 'D'}
]

n = len(transactions)
support_a = sum('A' in t for t in transactions) / n
support_b = sum('B' in t for t in transactions) / n
support_ab = sum(('A' in t and 'B' in t) for t in transactions) / n

confidence = support_ab / support_a
lift = confidence / support_b

print('support(A,B):', round(support_ab, 4))
print('confidence(A->B):', round(confidence, 4))
print('lift(A->B):', round(lift, 4))

if lift > 1:
    print('해석: A 구매는 B 구매 가능성을 평균보다 높이는 양의 연관을 보입니다.')

[문제 4: ADP 서술형] K-Means 결과 실루엣 점수가 0.46이고, 3개 군집이 각각 소형/중형/대형 고객군으로 해석된다고 가정합니다. 군집화 선택 이유, 결과 해석, 한계를 5문장 이내로 작성하세요.

본 문제는 정답 라벨 없이 고객의 유사한 특성을 기준으로 집단을 구분하는 것이 목적이므로 K-Means 군집화를 사용하였다.
K-Means는 거리 기반 방법이므로 변수 단위 차이를 제거하기 위해 표준화를 수행하였다.
실루엣 점수 0.46은 군집 구조가 어느 정도 존재하지만 매우 뚜렷하다고 보기는 어려운 수준으로 해석된다.
군집별 평균을 비교한 결과 소형, 중형, 대형 고객군으로 해석 가능하며, 마케팅 전략 차별화에 활용할 수 있다.
다만 군집 수와 초기값, 변수 선택에 따라 결과가 달라질 수 있으므로 추가 검증과 업무 전문가의 해석이 필요하다.
